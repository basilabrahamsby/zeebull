# 🚀 Quick Start Guide - Orchid Employee App

## 📱 **How to Use the App**

### **1. Login**
- Email: `housekeeping@orchid.com`
- Password: `1234`
- Click "LOGIN"
- ✅ Auto-login on next app start

---

### **2. Housekeeping Staff Workflow**

#### **A. Start Your Day**
1. Open app → Auto-login to Dashboard
2. Click **Clock In** (from drawer → Attendance & Salary)
3. Return to Dashboard

#### **B. Clean Rooms**
1. See "URGENT - Clean Now" section
2. Find a dirty room
3. Click "Start Cleaning"
4. Clean the room
5. Click "Mark Clean"
6. ✅ Completed count increases

#### **C. Audit Minibar (After Guest Checkout)**
1. Find room in urgent section
2. Click "Start Cleaning"
3. Click "Audit" button
4. Count consumed items:
   - Click "+" for each consumed item
   - Example: Water (3), Coke (2)
5. Click "Submit Consumption"
6. ✅ Charges added to guest bill

#### **D. Handle Service Requests**
1. Click orange FAB (Service Requests icon)
2. See all pending requests
3. Find a request (e.g., "Room 305 - Towels")
4. Click "Start Working"
5. Deliver items
6. Click "Mark Complete"
7. ✅ Request completed

#### **E. Report Damage**
1. From Room List, find the room
2. Click "Report Damage"
3. Select category (e.g., "Furniture")
4. Enter description ("Broken chair leg")
5. Click "Camera" or "Gallery"
6. Take/select photos (up to 5)
7. Click "Submit Damage Report"
8. ✅ Damage charged to guest

#### **F. End Your Day**
1. Open drawer → Attendance & Salary
2. Click "Clock Out"
3. ✅ Hours logged

---

### **3. Leave Management**

#### **A. Request Leave**
1. Open drawer → Attendance & Salary
2. Scroll down (or add button to navigate)
3. Navigate to Leave Request screen
4. Select leave type (Sick/Casual/Earned)
5. Pick start date
6. Pick end date
7. Enter reason
8. Click "Submit Request"
9. ✅ Request sent to manager

#### **B. Check Leave Status**
1. Navigate to Leave History screen
2. See all your leave requests
3. Filter by status:
   - Pending (Orange)
   - Approved (Green)
   - Rejected (Red)
4. View approval/rejection details

---

### **4. Navigation**

#### **Using the Drawer**
1. Tap hamburger menu (top-left)
2. See menu:
   - Dashboard
   - My Rooms
   - Service Requests
   - Attendance & Salary
   - Settings
   - Logout

#### **Using Floating Action Buttons**
- **Orange FAB**: Service Requests
- **Blue FAB**: All Rooms

## 🔔 **Status Indicators**

### **Room Status Colors**
- 🔴 **Red** = Dirty (needs cleaning)
- 🟠 **Orange** = Cleaning (in progress)
- 🟢 **Green** = Clean (ready)
- 🔵 **Blue** = Occupied (guest present)

### **Request Priority Colors**
- 🔴 **Red** = Urgent

- ⚪ **Grey** = Low

### **Leave Status Colors**
- 🟠 **Orange** = Pending
- 🟢 **Green** = Approved

## 🆘 **Troubleshooting**

### **Can't Login**
- Check email/password (case-sensitive)
- Ensure network connection
- Try: `housekeeping@orchid.com` / `1234`

### **Auto-Login Not Working**
- Clear app data
- Login again
- Token might be expired (24 hours)

### **Photos Not Uploading**
- Grant camera/storage permissions
- On web, use file picker instead
- Check photo size (max 5MB recommended)

### **Data Not Updating**
- Pull to refresh
- Expected behavior (mock data)
- Will work after backend integration
